# Something-Client-Mac
A mac ui client based on minecraft hack client looks.

This project is inspired by the Jackel client, with multiple features like cursor trail, windows trail, slapping the computer to trigger totem pops and a lot more, this is a full enhancement to the mac ui.

This is my first time making a project myself and first time coding, using git(hub), and a lot of things are new to me. I hope i can learn in the process, and in the way help others as well.
