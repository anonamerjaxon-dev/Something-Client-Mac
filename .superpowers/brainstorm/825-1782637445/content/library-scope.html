How should the system-wide library work?

A library that any app can import to add cursor trail functionality.

A

Swift Package

Distributed via SPM (Swift Package Manager). Any app adds it as a dependency. Recommended.

B

Framework Bundle

A .framework bundle that gets embedded in any app. More manual but very portable.

C

XCFramework

Apple's modern binary framework format. Works on macOS, iOS, and simulators. Best compatibility.
