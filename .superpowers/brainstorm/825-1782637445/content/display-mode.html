How should the trail appear on screen?

This determines how the trail is rendered on the macOS desktop. Different approaches have different trade-offs.

A

Transparent Overlay Window

A borderless, transparent window that floats above everything. The trail is drawn on this window. Works across all apps. Recommended.

B

Replace System Cursor

Hide the real cursor and draw a custom cursor + trail. More complex but feels native. Requires accessibility permissions.

C

Attach to Specific Window

Trail only appears over a specific app window (like a game or presentation). Good for targeted use cases.

D

Screen Recording Overlay

Draw the trail as part of a screen recording/stream overlay. Useful for content creators.
